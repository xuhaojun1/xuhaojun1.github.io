# Mathias's Homepage

My personal academic homepage, built with [Academic Pages](https://github.com/academicpages/academicpages.github.io) (Jekyll).

**Live site:** [https://xuhaojun1.github.io](https://xuhaojun1.github.io)

## Local Development

```bash
bundle install
bundle exec jekyll serve -l -H localhost
```

Then visit `http://localhost:4000`.
