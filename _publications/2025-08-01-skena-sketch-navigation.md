We propose the first sketch-based embodied visual navigation task and introduce the SkeNavigator framework for navigating unseen environments using abstract hand-drawn maps.

**Key Contributions:**
- Defined the first sketch-based embodied visual navigation task and built a fully automated data generation pipeline, producing the large-scale SoR dataset with 54K data samples.
- Proposed the SkeNavigator framework with a Ray-based Map Descriptor (RMD) for extracting abstract sketch features and a Dual-graph Alignment Goal Predictor (DAGP) module for aligning sketch semantics with real-time self-built map features.
- Achieved 105% improvement in navigation success rate (SPL) on high-abstraction validation sets of unseen environments compared to existing state-of-the-art methods.
